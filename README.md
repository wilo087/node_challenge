# Node.js Challenge

## Requirements
```sh
$ docker -v
# Docker version 20.10.24, build 297e128
```

## Getting started
Setup the environment
```sh
$ cp .env.example .env
```

Run the services
```sh
$ docker compose up

# if you want to run in background
$ docker compose up -d
```

You endpoint should be available at `http://localhost:3000`
